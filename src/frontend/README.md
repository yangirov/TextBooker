# Frontend

Make sure you have a [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
